Sobre el software

Destinatario del Software:

El software educativo está destinado a la captación laboras determinada en esto se realiza la distinta capacitación y posterior empleo en periodo de prueba.

También es necesario tener cierto nivel de conocimiento en redes de información para el uso de este software

Tecnologías usadas en el software:

Este software está construido con el motor grafico de videojuegos Unity 3d junto con el lenguaje de programación c#.

Gracias a unity pudimos realizar un software con características lúdicas permitiendo al usuario tener mayor interacción con el mismo a través de actividades prácticas.
